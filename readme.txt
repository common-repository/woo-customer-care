== Description ==

**WPC Customer Care** is a complete customer care system for WooCommerce. You can self-assign or assign an order to another one. When the order was assigned to one manager, another one can't access and make any change for this order. All actions will be logged and we can easily in track who is caring the order.

= Features =

- Auto-assign
- Self-assign and assign to another one
- Prevent access when someone is caring an order
- Track all changes

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Customer Care", Install & Activate it
4. Now when you open the WooCommerce/Order page you can see this feature
